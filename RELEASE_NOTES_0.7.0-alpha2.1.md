# FibroChat v0.7.0-alpha2.1

Мобильный hotfix интерфейса чата.

## Исправлено

- единая высота приложения через Visual Viewport;
- прокрутка ограничена списком сообщений;
- поле ввода остаётся доступным при открытой клавиатуре;
- в открытом мобильном чате скрывается нижняя навигация и освобождается вся высота;
- устранён вызов `scrollIntoView()` для поля ввода, который сдвигал весь интерфейс;
- добавлена корректная поддержка safe-area;
- обновлён кэш Service Worker.

## Проверки

- `npm run check`;
- `npm run smoke`.
