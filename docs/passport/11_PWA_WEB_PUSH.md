# PWA и Web Push

FibroChat v0.4.3 регистрирует Service Worker, устанавливается как PWA и хранит Web Push подписки в PostgreSQL.

## Требования
- Публичный HTTPS-адрес.
- Пользователь должен нажать кнопку включения уведомлений.
- На iPhone/iPad (iOS/iPadOS 16.4+) Web Push работает для приложения, добавленного на экран «Домой».
- Текст сообщений в push не передаётся: уведомление сообщает только о новом защищённом сообщении.

VAPID-ключи создаются автоматически при первом запуске и сохраняются в конфигурации сети PostgreSQL.
