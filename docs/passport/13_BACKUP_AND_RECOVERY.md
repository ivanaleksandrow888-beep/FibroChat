# Резервное копирование и восстановление

Перед крупным релизом сохраняются: исходный ZIP, Git-коммит, экспорт PostgreSQL и зашифрованная копия сети из панели суперадминистратора. Откат кода выполняется через предыдущий Git-коммит/ZIP и redeploy. Откат схемы БД нельзя выполнять удалением базы; для него нужна отдельная обратимая миграция или восстановление дампа.
