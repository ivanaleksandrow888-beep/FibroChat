# Текущее состояние

## Работает
- Docker/Coolify/HTTPS.
- PostgreSQL и автоматические миграции.
- Первичная инициализация суперадминистратора.
- Регистрация по инвайту, подтверждение пользователей, назначение администраторов.
- Защищённые сообщения, сессии, устройства, уведомления, поддержка.
- PWA, Service Worker, Web Push серверная база.

## Версия 0.5.0
- отдельные разделы интерфейса;
- восстановление сессии после F5;
- шестизначный локальный PIN;
- управляемая блокировка;
- обновлённый кэш Service Worker.

## Требует проверки после деплоя
- F5 после обычного входа;
- закрытие/открытие PWA с PIN;
- push в Android Chrome;
- push в iOS PWA;
- роли администратора и суперадминистратора в новой навигации.
