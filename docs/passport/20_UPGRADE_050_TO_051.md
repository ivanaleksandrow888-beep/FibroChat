# Обновление 0.5.0 → 0.5.1

1. Создать резервную копию PostgreSQL и каталога проекта.
2. Заменить файлы проекта содержимым архива 0.5.1.
3. Выполнить `npm ci` или пересобрать Docker-образ без кэша.
4. Запустить приложение. Сервер автоматически применит миграцию `003_private_contacts_fibro_id.sql`.
5. Проверить профиль: у каждого аккаунта должен отображаться Fibro ID.
6. Проверить контакты под обычным пользователем, администратором и суперадминистратором.

Существующим пользователям Fibro ID назначается автоматически. Связь с пригласившим восстанавливается из истории использованных инвайтов, если в старых данных сохранились `createdBy` и `usedByNickname`.
