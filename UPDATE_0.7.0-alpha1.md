# Обновление до v0.7.0-alpha1

1. Сделайте резервную копию базы и каталога `data`.
2. Замените файлы приложения.
3. Выполните `npm install`.
4. Перезапустите сервер. PostgreSQL-миграция применяется автоматически.
5. Убедитесь, что reverse proxy допускает тело запроса не менее 11 МБ.
6. Для другого лимита задайте `MAX_ATTACHMENT_BYTES` в байтах.

Каталог `data/attachments` не должен публиковаться как статический.
