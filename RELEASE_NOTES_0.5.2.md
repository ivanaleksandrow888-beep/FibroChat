# FibroChat v0.5.2 — Profile & Privacy

## Главное

- Полноценная страница профиля пользователя.
- Отображаемое имя, описание и аватар PNG/JPEG/WebP до 250 КБ.
- Неизменяемый Fibro ID, QR-код и ссылка-приглашение.
- Настройки приватности профиля, первого сообщения, поиска по Fibro ID и добавления в контакты.
- Удаление контакта, блокировка, разблокировка и список заблокированных.
- Профили контактов учитывают взаимные блокировки и настройки видимости.
- Сервер запрещает сообщения и добавление по Fibro ID, если это ограничено приватностью.

## API

- `GET /api/profile`
- `PUT /api/profile`
- `GET /api/profiles/:id`
- `GET /api/blocked`
- `POST /api/contacts/:id/delete`
- `POST /api/contacts/:id/block`
- `POST /api/contacts/:id/unblock`

## Совместимость

Миграция выполняется автоматически. Существующим пользователям назначаются безопасные значения профиля и приватности без изменения Fibro ID и ключей.
