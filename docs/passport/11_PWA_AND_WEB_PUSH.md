# PWA и Web Push

Service Worker кэширует оболочку версии 0.5.0 по стратегии network-first. Push-подписка хранится в PostgreSQL. На iOS Web Push требует iOS/iPadOS 16.4+ и запуск приложения после добавления на экран «Домой». Разрешение на уведомления должно запрашиваться после пользовательского действия.
