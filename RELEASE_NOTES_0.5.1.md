# FibroChat v0.5.1 — Private Contacts & Fibro ID

## Готово

- Общий список пользователей удалён из пользовательского API контактов.
- Добавлена ролевая видимость контактов.
- Добавлен постоянный уникальный Fibro ID.
- Добавление контакта работает только по полному точному Fibro ID.
- Частичный поиск и подсказки отсутствуют.
- Добавлена таблица `contacts` и автоматическая миграция.
- API сообщений проверяет право на контакт.
- Контакты, добавленные по Fibro ID, создаются взаимно.
- Паспорт проекта, changelog и инструкция обновления актуализированы.

## Проверки

- `npm ci` — успешно.
- `npm run check` — успешно.
- Интеграционный тест JSON fallback — успешно: регистрация владельца, назначение администратора, приглашение пользователя, ролевая видимость, точное добавление по Fibro ID, отказ частичному ID.
- PostgreSQL-миграция подготовлена, но реальный PostgreSQL и Docker в среде сборки не запускались.
