# Архитектура

```text
Браузер / установленная PWA
  -> HTTPS
Coolify reverse proxy
  -> Docker-контейнер FibroChat (Node.js, порт 3000)
  -> DATABASE_URL
PostgreSQL (отдельный ресурс и постоянный том)
```

GitHub хранит код. Coolify скачивает ветку `main`, собирает `Dockerfile` и заменяет только контейнер приложения. PostgreSQL не должен удаляться при redeploy.

Клиент пока использует `app.js` как совместимое функциональное ядро и `ui-shell.js` как отдельный модуль маршрутизации/компоновки. Это первый безопасный шаг к полной модульности без изменения API.
