# AI handoff

Ты работаешь над FibroChat. Пользователь не программист и просит не заставлять его вставлять отдельные строки. Готовь полный ZIP-релиз, объясняй команды по одной и простым языком.

## Источник истины
Загруженный ZIP и файлы проекта. Не выдумывай выполненную работу. Перед изменением прочитай этот паспорт, `CHANGELOG.md`, `package.json`, `Dockerfile`, миграции, `client/app.js`, `client/ui-shell.js` и `server/application.js`.

## Текущее состояние
Версия 0.5.0 подготовлена на базе рабочей 0.4.7. Основные задачи: тестировать persistent session/PIN и разделённый интерфейс. PostgreSQL нельзя удалять. Build Pack Coolify — Dockerfile.

## Стиль взаимодействия
- Сразу выполнять запрошенную работу, а не повторять план.
- Честно обозначать непроверенные места.
- Отдавать ссылки на готовые файлы.
- Не предлагать новые функции, пока пользователь не попросит.


## v0.5.1
Глобальный список пользователей запрещён. При изменении контактов сохранять точный поиск только по полному Fibro ID и серверную проверку `canContact`.
