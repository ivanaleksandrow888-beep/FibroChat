# Обновление FibroChat 0.5.1 → 0.5.2

1. Сделайте резервную копию базы PostgreSQL и постоянного каталога данных.
2. Распакуйте архив v0.5.2 поверх рабочей копии проекта.
3. Выполните `npm ci`.
4. Выполните `npm run check` и `npm run smoke`.
5. Закоммитьте и отправьте изменения в ветку `main`.
6. В Coolify запустите **Redeploy**. При наличии старого кэша выберите **Force Rebuild**.
7. Проверьте вход, страницу профиля, QR-код, сохранение приватности, удаление и блокировку контакта.

Миграция `004_profile_privacy.sql` применяется сервером автоматически при старте PostgreSQL-режима.

## После деплоя
Статические HTML/JS/CSS-файлы в этой сборке отдаются с `Cache-Control: no-store`. После первого открытия обновлённой версии перезагрузите страницу один раз. В разделе «Профиль» должна отображаться отметка `v0.5.2`.
